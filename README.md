# hermannbjorgvin's Claude Plugins

A collection of plugins for [Claude Code](https://claude.ai/claude-code).

## Installation

```bash
/plugin marketplace add HermannBjorgvin/claude-pauser
/plugin install <plugin-name>@hermannbjorgvin
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [claude-pauser](./claude-pauser/) | Pause your Apple TV when Claude needs attention, resume when working |
